# 🏢 Apex Scaling Agency Portal
Next.js landing page for Apex Scaling Agency showcasing AI automation and business scaling solutions.

Status: 90% complete — LAUNCH

## 🚀 Features
*   **Spline 3D Scene Integration**: Interactive floating 3D design components.
*   **GSAP Timeline Animations**: High-performance scrolling and loading transitions.
*   **Tailwind CSS v4 & React 19**: Ultra-modern, responsive styling layout.
*   **Lead Capture & Validation**: Secure form interface built with class-variance-authority and Zod.

## 🛠️ Tech Stack
*   **Framework**: Next.js 16 (React 19)
*   **Libraries**: GSAP, Spline, Framer Motion, Lucide React
*   **Styling**: Tailwind CSS v4

## 🔮 What's Left to Ship
We need to bind the custom domain DNS routing and set up environment variable tokens for the Resend client in a production environment. This is the final step to make the contact form fully operational.

## 📦 How to Run Locally
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  Open `http://localhost:3000` in your browser.
